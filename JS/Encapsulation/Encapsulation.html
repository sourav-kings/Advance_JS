Encapsulation

  • Put all the inner workings of an object inside that object
  • To implement encapsulation in JavaScript, we have to define the core methods and properties on that object.
  • Best pattern for encapsulation in JavaScript: the Combination Constructor/Prototype Pattern.
    • Why we need it? When multiple objects of similar kind i.e. similar properties and methods.
    • How it works? Encapsulate these functionalities inside a function, and use this Function's constructor to create objects.
  • When to prefer Object Literal pattern? When purpose is to store some data, and it is the only object of its kind.
