• prototype:- It is a property of the function. It is an object named "__proto__". Under this property we've construtor, and we can assign custom properties.
  • Use of "this":- It specifies that these properties will be unique to every instance of the "User" object.
  • IMP - This vs. Prototype:- Both are almost same, excpet Prototype one is relatively faster for huge number of objects. Check speed test
    1.> Actual definations of the properties declared using "this" are private and can be changed only by function's object instance. This instance invoked change is limited to that instance only.
    2.> However, actual definations of the properties declared using "prototype" can be changed with or without object instance. And that copy of the prototype is shared by all the object instances.
